
# Project Title

React Application


## Tech Stack

**Client:** React,  ReactPrime, PrimeFlex




## Run Locally

Clone the project

```bash
  https://github.com/Sravan-P-Thankappan/paripoorna.git
```

Go to the project directory

```bash
  cd client
  
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm  start
```
